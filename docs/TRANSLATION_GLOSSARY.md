## Translation Rules

1. **Regional variants:** Mexican/Latin American Spanish, Brazilian Portuguese
2. **Register:** Professional but warm. "Usted" for system messages, "tu" acceptable for friendly UI text.
3. **Gender:** Use gender-appropriate forms for roles (Maestro/Maestra). When generic, use "Maestro/a" or masculine default.
4. **Placeholders:** Preserve `%s`, `%d`, `%1$s` etc. exactly
5. **HTML:** Preserve all tags exactly
6. **UI labels:** Keep concise (buttons, badges, menu items)
7. **Navigation buttons:** Use full forms (Siguiente, not Sig; Anterior, not Ant)
8. **Coach:** Always keep as "Coach" in both languages — never "Entrenador" or "Treinador"
9. **When unsure:** Add `## REVIEW:` comment in the .po file

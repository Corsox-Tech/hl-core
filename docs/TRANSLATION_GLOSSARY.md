## Translation Rules

1. **Regional variants:** Mexican/Latin American Spanish, Brazilian Portuguese
2. **Register:** Professional but warm. "Usted" for system messages, "tu" acceptable for friendly UI text.
3. **Placeholders:** Preserve `%s`, `%d`, `%1$s` etc. exactly
4. **HTML:** Preserve all tags exactly
5. **UI labels:** Keep concise (buttons, badges, menu items)
6. **When unsure:** Add `## REVIEW:` comment in the .po file
